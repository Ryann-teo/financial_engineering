1. Build specification

The capstone strategy:

  • Universe: 6 G10 currencies, 5 government bonds, 8 equity index futures, 6 commodities. 25 instruments across 4 asset classes.
  • Signals: trend, carry, value, macro growth surprise, macro inflation surprise. 5 signals.
  • Forecast: equal-weighted composite signal across signals (conceptual parity).
  • Optimisation: sigmoid sizing on the composite.
  • Risk: per-asset vol targeting at 10%, then asset-class ERC across the 4 buckets.
  • Drawdown: MinMax overlay at 12% threshold, $\kappa = 15$.
  • Costs: 5 bp DM, 15 bp EM and commodities.
  • Rebalance: monthly, with 50 bp drift band.

2. Universe and data

Synthetic data for the demo. In production the macro signals come from the JPMaQS macro-quantamental dataset (Macrosynergy product).

3. Signal stack

For each instrument, compute 5 signals:

  • Trend. 12-month log return minus current value (TSMOM).
  • Carry. Yield differential (FX), futures roll yield (commodity), term-spread (rates), dividend yield (equity).
  • Value. 5-year mean reversion z-score.
  • Macro growth. Real GDP nowcast minus trend, point-in-time.
  • Macro inflation. CPI YoY minus central bank target.

Each signal is z-scored within asset class and capped at ±2.

4. Composite forecast

Equal-weight the 5 signal z-scores into a single composite per instrument. No optimisation. Conceptual parity.

5. Risk and execution overlay

  1. Sigmoid sizing of composite to position $\in [-2, 2]$.
  2. Per-asset vol target at 10% (lagged EWMA, 22-day half-life).
  3. Within asset class, equal weight (alternative: ERC).
  4. Across asset classes, equal risk contribution.
  5. Strategy-level vol target at 10%.
  6. MinMax drawdown overlay.
  7. Cost layer: 5 bp DM, 15 bp EM/commodities, applied to monthly turnover.

6b. Deep dive: performance attribution by signal family

Once the capstone runs, the next question is always: where did the PnL come from? Performance attribution decomposes total return into contributions from each signal family (trend, carry, value, growth, inflation), each asset class, and the risk overlay (vol target, drawdown control).

Per-signal-family attribution

For each signal $k$, compute the portfolio return that would have resulted if only signal $k$ had been used (with the same risk overlay). The total composite return is approximately the sum of per-signal contributions, with cross-terms captured in an "interaction" residual.

7. Coding exercises (extras)

Exercise A: Build the capstone in code

class Capstone:
    def __init__(self, config):
        self.config = config
    def universe(self): ...
    def signals(self): ...  # trend, carry, value, growth, inflation
    def composite(self):
        return self.signals().mean(axis=1)
    def positions(self):
        return np.tanh(self.composite())
    def vol_target(self, positions):
        sigma = self.realised_vol().shift(1)
        return positions * (self.config.target_vol / sigma).clip(upper=3)
    def drawdown_overlay(self, ret):
        eq = (1 + ret).cumprod()
        peak = np.maximum.accumulate(eq)
        dd = 1 - eq / peak
        lev = np.where(dd < self.config.dd_thresh, 1, np.exp(-self.config.kappa * (dd - self.config.dd_thresh)))
        return ret * lev
    def costs(self, weights):
        turnover = (weights.diff().abs().sum(axis=1)).mean() * 12
        return turnover * self.config.cost_bp / 10000
    def run(self):
        sigs = self.signals()
        comp = self.composite()
        pos = self.positions()
        pos_vt = self.vol_target(pos)
        ret = (pos_vt * self.returns()).sum(axis=1)
        ret = self.drawdown_overlay(ret)
        return ret - self.costs(pos_vt) / 252

8. Self test

Question 1, conceptual

Why is the Macrosynergy flagship strategy correlation to S&P 500 only 10 to 20% despite using equity indices? Hint: long-short cross-sectional structure and macro signal diversity.

Question 2, coding

Question 3, numerical

For the capstone with target vol 10%, max DD 12%, cost 5 bp, IC 0.10 across 25 instruments, what would you expect for net Sharpe?

Answer: Roughly 0.8 to 1.2 (slightly below the in-sample backtest because of cost and DSR adjustments).
