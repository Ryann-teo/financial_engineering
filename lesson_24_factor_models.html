0. Reading the controls

Number of factors $K$

How many systematic factors. Single-index = 1. FF5 = 5. BARRA = 50 to 80 (industries + styles). Statistical PCA: choose K via Bai-Ng or Marcenko-Pastur cutoff.

Idiosyncratic share

What fraction of single-name variance is unexplained by factors. Typical: 30 to 60% for single stocks under FF5.

Factor type

Statistical (PCA), fundamental (BARRA), macroeconomic (Chen-Roll-Ross). Different philosophies, different applications.

1. The single-index model

Sharpe (1963): every asset has a beta to the market.

$$r_i = \alpha_i + \beta_i r_m + \epsilon_i$$

Properties:

  • Total risk: $\sigma_i^2 = \beta_i^2 \sigma_m^2 + \sigma_{\epsilon_i}^2$.
  • Systematic risk: $\beta_i^2 \sigma_m^2$.
  • Idiosyncratic risk: $\sigma_{\epsilon_i}^2$.
  • Covariance: $\sigma_{ij} = \beta_i \beta_j \sigma_m^2$.

The single-index model collapses the $N \times N$ covariance matrix into $2N + 1$ parameters. Massive parsimony for large $N$.

2. Multi-factor: Fama-French and Carhart

The CAPM beta is not enough. Fama-French (1993) added size (SMB) and value (HML). Carhart (1997) added momentum (UMD). Fama-French 5-factor (2015) added profitability (RMW) and investment (CMA).

$$r_i - r_f = \alpha_i + \beta_{i,M} (r_M - r_f) + \beta_{i,\text{SMB}} \text{SMB} + \beta_{i,\text{HML}} \text{HML} + \ldots + \epsilon_i$$

The $\alpha_i$ is the "true alpha" after factor exposures. Most managers' alpha shrinks toward zero when measured against FF5+. The factor model is a yardstick: are you actually generating returns above and beyond the systematic styles?

3. Statistical factors via PCA

Connor and Korajczyk (1986): for $N$ assets and $T$ observations, the first few principal components of the return covariance matrix are the dominant factors.

Procedure:

  1. Compute $\Sigma$ from sample.
  2. Eigendecompose: $\Sigma = V \Lambda V^\top$.
  3. Choose $K$ via Bai-Ng or visual inspection of eigenvalue spectrum.
  4. Factor returns: $f_t = V_K^\top r_t$.
  5. Factor loadings: $B = V_K$.

Statistical factors have no economic interpretation; they are abstract directions in return space. The first PC almost always looks like the market.

4. Fundamental factors: BARRA

BARRA (Rosenberg 1974) defined factors by observable characteristics:

  • Style factors: size, value, growth, momentum, quality, low volatility.
  • Industry factors: 50+ GICS sub-industries.
  • Country and currency factors: for global universes.

Loadings come from observables (market cap, P/B, past returns, GICS code). Factor returns are estimated from cross-sectional regressions of $r_i$ on $B_i$ each period. Industry standard for equity risk models since the 1980s.

5. Risk decomposition for portfolios

Given the factor model $\Sigma = B \Omega_F B^\top + D$:

$$\sigma_p^2 = \underbrace{(B^\top w)^\top \Omega_F (B^\top w)}_{\text{factor risk}} + \underbrace{w^\top D w}_{\text{idiosyncratic risk}}$$

For a typical 30-stock long-only equity portfolio, factor risk is 70 to 90% of total. Idiosyncratic averages out by diversification, leaving factor exposures as the dominant risk.

Within factor risk, decompose further by factor:

$$\text{factor risk}_k = (B^\top w)_k \cdot (\Omega_F B^\top w)_k$$

This tells you which factors carry the portfolio's risk. Typical finding for long-only equity: 50 to 80% market, 5 to 15% size, 5 to 15% value, etc.

6. Mathematical companion

6.1 Single-index model derivation

Regress $r_i = \alpha_i + \beta_i r_m + \epsilon_i$ via OLS. The estimated $\hat \beta_i$ is $\text{Cov}(r_i, r_m) / \text{Var}(r_m)$. This is also the slope of the line through the data.

6.2 Bai-Ng criterion for choosing K

Information criterion that trades off explained variance (more factors better) against parsimony (fewer factors better). Choose K to minimise $\log(\text{SSR}_K) + K \cdot \frac{N + T}{NT} \log\left(\frac{NT}{N+T}\right)$.

6.3 BARRA cross-sectional regression

At each date $t$: stack assets, run $r_t = B f_t + \epsilon_t$ where $B$ is fixed (loadings from observables) and $f_t$ are estimated factor returns. Standard OLS. Repeat each period.

6.4 Idiosyncratic risk diversification

For an equal-weight portfolio of $N$ assets with average idiosyncratic vol $\sigma_\epsilon$, portfolio idiosyncratic vol is $\sigma_\epsilon / \sqrt{N}$ (assuming uncorrelated residuals). For $N = 30$: idiosyncratic vol is 18% of single-name idiosyncratic vol. Most of the diversification benefit is in idiosyncratic risk.

6.5 Macroeconomic factor models

Chen, Roll, Ross (1986) show that macro factors (industrial production growth, inflation surprise, term-spread, default-spread) have non-trivial cross-asset exposures. Predates BARRA's empirical-loading approach.

7. Coding exercises (extras)

Exercise A: Single-index regression

def single_index(returns: pd.DataFrame, market: pd.Series) -> dict:
    betas, alphas = {}, {}
    for col in returns.columns:
        cov = returns[col].cov(market)
        beta = cov / market.var()
        alpha = returns[col].mean() - beta * market.mean()
        betas[col] = beta
        alphas[col] = alpha
    return {'betas': betas, 'alphas': alphas}

Exercise B: Fama-French 5-factor regression

import statsmodels.api as sm
def ff5_regression(portfolio_returns, factor_returns):
    X = sm.add_constant(factor_returns)
    model = sm.OLS(portfolio_returns, X).fit()
    return model.params, model.tvalues

Exercise D: Risk decomposition

def factor_risk_decomposition(w, B, Omega_F, D):
    factor_exp = B.T @ w
    factor_var = factor_exp @ Omega_F @ factor_exp
    idio_var = w @ D @ w
    total_var = factor_var + idio_var
    return {
        'factor_var': factor_var,
        'idio_var': idio_var,
        'factor_share': factor_var / total_var
    }
