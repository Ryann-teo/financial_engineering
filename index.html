About this curriculum

An eight-week journey that takes a learner from "what is a return?" through advanced portfolio optimisation: Markowitz, risk parity, Black-Litterman, CVaR, hierarchical risk parity, vol targeting, factor models, and a cross-asset macro capstone.

Every lesson is a self-contained interactive HTML page with KaTeX math, Plotly demos, embedded code exercises, and "in practice" boxes connecting the abstract math to a real Macrosynergy strategy where applicable.

No installation required to read the lessons. Open any HTML file in your browser. For the coding exercises, see module_a_exercises.ipynb.
